# Globos centras "Šv. Gertrūda" - [Website](http://gcsg.lt/)

## Pre-requisites

-  Install Node.js version 20.15.0

## Getting started

-  Clone the repository

```
git clone <url>
```

```
cd <project_name>
npm install
```

-  To start development at localhost, run:

```
npm run dev
```

-  To run eslint & typescript types tests, run:

```
npm run test
```
